## 功能拆开来说说

### 缩写展开：最核心的那个

输入缩写，自动替换成你预设的内容。内容可以是：

- 纯文本（邮件签名、套话模板）
- 富文本（带格式的内容）
- 代码片段
- 动态内容（插入当前日期、剪贴板内容、光标占位符）

比如客服每天要发一百遍的"您好，感谢您的耐心等待……"，设一个缩写 `nhs`，以后两秒搞定。

### 脚本执行：真的会跑代码

这是让很多人眼前一亮的功能。片段里可以内嵌脚本，支持 JavaScript、Python、Bash Shell、AppleScript——触发缩写的时候，脚本实时跑，把结果插进光标位置。

想要每次输出当前 Git commit hash？想要自动生成一段带时间戳的注释？写个脚本挂上去就行。

### 自定义模板：填空式输入

触发某些片段的时候，会弹出一个小表单让你填写变量，填完按确认，完整内容生成出来。适合那种"结构固定、细节每次不同"的场景，比如 bug 报告模板、代码 review 注释格式。

### 跨平台云同步

支持 Windows 和 macOS，数据可以同步到 GitHub Gist 或码云。这意味着你在公司 Mac 上维护的那几百个片段，回家打开 Windows 本，一模一样。

### 截图与剪贴板

新版 CodeExpander Pro 还集成了截图标注（支持模糊、马赛克、高亮、Pin 固定悬浮）和剪贴板历史功能。付费版剪贴板历史保存 100 条，不用再为"刚才复制的那段去哪了"抓耳挠腮。
